Event Landing Page

A modern and responsive Event Landing Page built using HTML, CSS, and JavaScript. This project showcases an engaging event website with a clean layout, responsive design, and interactive elements to provide visitors with essential event information.

✨ Features

- Responsive design for desktop and mobile devices
- Modern and visually appealing UI
- Hero section with event highlights
- Event information and schedule
- Interactive navigation
- Smooth scrolling and animations
- Call-to-action buttons
- Lightweight and fast performance

🛠️ Built With

- HTML5
- CSS3
- JavaScript (Vanilla JS)

🎯 Purpose

This project was developed to strengthen my front-end development skills by creating a professional event landing page. It demonstrates responsive web design, modern UI implementation, and JavaScript interactivity while serving as a portfolio project.

🚀 Getting Started

1. Clone this repository.
2. Open "index.html" in your preferred web browser.
3. Explore the landing page.
